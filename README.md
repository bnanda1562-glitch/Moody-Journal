# 🧠 Mood Journal AI

An AI-powered mood journaling application that helps users track emotions, maintain journaling streaks, and receive personalized AI insights through text and voice entries.

## ✨ Features

- 🔐 JWT Authentication
- 📝 Text & Voice Journaling
- 🤖 AI Emotion Analysis (Google Gemini)
- 📊 Mood Analytics Dashboard
- 🔔 Real-time Notifications (Socket.IO)
- 🔥 Streak Tracking & Achievement Badges

## 🛠️ Tech Stack

- **Frontend:** React.js, Chart.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT
- **AI:** Google Gemini API
- **Realtime:** Socket.IO
